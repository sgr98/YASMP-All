# YASMP

## My Environment: 

* node --version: v14.12.0
* npm --version: 6.14.8 
* create-react-app: 5.0.1

<br>

## Steps to run

Clone repository and navigate to YASMP-All directory on your terminal. Then:


For setting up the environment:
```
1. cd meta
2. node meta.mjs
3. cd ../yasmp
4. npm i
5. cd ../yasmpseer
6. npm i
```

For running the environment:
```
1. Open 2 terminals
2. On 1st terminal
    1. cd yasmp
    2. npm run uconc
3. On 2nd terminal
    1. cd ../yasmpseer
    2. npm run uconc
```
